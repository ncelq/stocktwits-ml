# StockTwits Machine Learning

## Prerequisite
- Apache Spark 2.2

## Configuration
The spark home is set in src/main/resources/application.properties
```
spark.home=/<spark home>/Software/spark
```

## Build
```
mvn clean install
```

## Run
```
mvn sprint-boot:run
```

## Test
Go to http://localhost:8080/index.html to test the application
